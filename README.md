#  Eastern Mechanic Simulator - 3D Car Mechanic Simulator (Unity)

> [!CAUTION]
> **WORK IN PROGRESS (Fejlesztés alatt)**
> Ez a projekt jelenleg is aktív fejlesztés alatt áll. A repository a jelenlegi kódbázist és a mechanikák állapotát tükrözi.

## 🇭u Magyar leírás
Egy komplex, objektumorientált (OOP) alapokon nyugvó autószerelő szimulációs játékprojekt, Unity 3D játékmotorban (C#) fejlesztve. 

###  Kiemelt fejlesztési mechanikák:
* **Valósághű 3D fizika:** Alkatrészek fizikai tárgykezelése (Rigidbody tömeg- és lendületcsillapítás, Raycasting alapú interakciók).
* **Moduláris járműrendszer:** Dinamikusan szerelhető, cserélhető és állapotfüggő alkatrészek rendszere.
* **Egyedi Save/Load rendszer:** Saját fejlesztésű, hibatűrő JSON alapú állapotmentő rendszer, amely a járművek és alkatrészek minden tulajdonságát és helyzetét képes elmenteni és visszatölteni.

---

## 🇬🇧 English Description
A complex, Object-Oriented Programming (OOP) based car mechanic simulation game project, developed in the Unity 3D Engine using C#.

###  Key Development Features:
* **Realistic 3D Physics:** Physical object handling of car parts (Rigidbody mass and drag configurations, Raycasting-based player interactions).
* **Modular Vehicle System:** Dynamically attachable, detachable, and state-dependent vehicle components.
* **Custom Save/Load Architecture:** A self-developed, fault-tolerant JSON-based state management system capable of serializing and deserializing the exact position and status of vehicles and all individual parts.
